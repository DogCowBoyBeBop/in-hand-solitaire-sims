# Solitaire Simulators

These are some programs I threw together to simulate in-hand solitaire games.

They're in bash, which is not ideal but works.

[1h3.sh](1h3.sh) simulates a variety of [_Accordian_-style games](https://www.metafilter.com/212541/More-productive-than-doomscrolling). [shoutoutsim.sh](shoutoutsim.sh) simulates the game Shout Out.
